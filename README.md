# Bib2spsh (和訳)

Bib2SPSH (BibTex To SpreadSheet) は数学の論文情報を（JSON に変換し、）スプレッドシートに書き出すツールです。
これを使用することで、あなたが閲覧した論文を記録し、それらを研究室の仲間や教授に共有することができます。
